# Hamza Virk - Personal Academic Website

This is the source code for Hamza Virk's personal academic website, showcasing his journey as an undergraduate mathematics student at Hofstra University.

## About

This website serves as a digital portfolio and academic presence for Hamza Virk, featuring:

- **Academic Profile**: Information about research interests and academic background
- **Research**: Current and future research projects in mathematics
- **Blog**: Thoughts and insights on mathematical concepts and academic life
- **CV**: Academic curriculum vitae and achievements
- **Contact**: Ways to connect and collaborate

## Website Features

- **Responsive Design**: Optimized for all devices
- **Academic Focus**: Tailored for academic and research presentation
- **Blog Integration**: Share mathematical insights and academic experiences
- **Project Showcase**: Display research projects and academic work
- **Publication Ready**: Prepared for future academic publications

## Local Development

To run this website locally:

1. **Prerequisites**: Ensure you have Ruby and Bundler installed
2. **Install Dependencies**: `bundle install`
3. **Start Server**: `bundle exec jekyll serve`
4. **View Website**: Open `http://localhost:4000` in your browser

## Deployment

This website is automatically deployed using GitHub Pages. Any changes pushed to the main branch will be automatically reflected on the live website.
